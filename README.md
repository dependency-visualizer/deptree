deptree
=======

## description
view dependency tree of your modules in browser.

## usage

    $ git clone https://github.com/leungwensen/deptree.git
    $ cd deptree
    $ npm install
    $ ./bin/deptree.js -d node_modules/pastry/src -a js-amd
