Welcome to homepage of deptree.

deptree is a tool to display the dependency tree of your modules, so that you can understand the connects between modules and dependency circle easier.

$ git clone https://github.com/leungwensen/deptree.git
$ cd deptree
$ npm install
$ ./bin/deptree.js -d node_modules/pastry/src -a js-amd
